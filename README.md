# 🔐 Cloud Security Audit Dashboard (Zero Trust Enabled)

## 🚀 Overview
A serverless cloud security monitoring system built on AWS with Zero Trust security principles.

## 🛠️ Tech Stack
- AWS Lambda
- API Gateway (IAM Auth)
- Amazon S3 (Private)
- EventBridge
- React.js
- CloudFront (CDN)

## 🔐 Security Features
- IAM-based API authentication (Zero Trust)
- No public access to backend resources
- Least privilege access control
- Secure API invocation using AWS Signature

## 📊 Features
- Real-time security audit dashboard
- Severity classification (High/Medium/Low)
- Filtering and visualization
- Email alerts for critical risks

## 🧪 Zero Trust Demo
- Unauthorized access → Blocked ❌
- Authorized IAM request → Allowed ✅
